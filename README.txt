1) Pour être en mesure d'utiliser l'application, il est conseillé de brancher un téléphone mobile Androide avec un fils à l'ordinateur. Ou s’il n’y a pas de téléphone mobile à la disposition, il y a la possibilité d'installer un émulateur pour pouvoir exécuter l'application. Une fois que cela est fait, il faut cliquer sur la petite flèche à côté du bouton vert exécuter avec le débogage. Il faut sélectionner le téléphone qui est branché à l'ordinateur ou l'émulateur qui a été installés au besoin. Par la suite il faut lancer l'application grâce au débogage ou simplement l'exécuter sans débogage, c'est au choix. L'application est maintenant installée sur le téléphone et elle s'est aussi exécutée.

2) Il y a maintenant la possibilité d'entrer des informations valides dans les champs puis faire suivant. Vous allez finir par arriver sur une page placeholder ou la vidéo tutoriel vas être inséré. (L’utilisateur est sauvegardé)

3) Une fois que vous avez créé un utilisateur et que vous être redirigé sur la page tutoriel. Vous pouvez continuer. Vous allez être sur une page pour créer un nouvel entrainement. Il y a une liste de tous les entrainements, vous pouvez scroller avec votre doigt. Par la suite, il suffit de sélectionner un entrainement et dans les 3 champs en dessous de mettre les informations que vous voulez. Vous pouvez appuyer sur Ajouter. Cela va permettre de garder votre exercice en mémoire dans la liste en dessous du bouton. Une fois tous vos exercices de créer, vous pouvez en sélectionner un et le supprimer si vous avez fait une erreur. Une fois que c'est terminer, vous avez juste à cliquer sur confirmer. Si vous voulez continuer sans sauvegarder votre entrainement vous pouvez appuyer sur annuler. Vous allez être sur la page dernier entrainement ou il y a la liste des exercices avec leurs données. De plus, il y a un bouton pour retourner sur la page pour créer un entrainement. Dans le haut il y a un menu qui permet de naviguer. Vous pouvez déplacer l’écran vers la droite ou vers la gauche ou simplement naviguer avec le menu. La prochaine page est la page de la liste d’exercices. Dans cette page vous pouvez retrouver la liste de tous les exercices, avec leur image, leur nom et les groupes musculaires. Si vous appuyez dessus vous allez avoir une fenêtre qui donne tous les détails. Il y a toujours le bouton pour créer un nouvel entrainement. La page suivante est la page statistique (progression). Vous pouvez sélectionner un moment et un exercice, les données vont être affichées dans le graphique. Pour terminer il y a la page modifier utilisateur qui comporte 2 boutons un pour modifier et l’autre pour supprimer toutes les données de l’utilisateur.
4) maintenant si vous fermez l'application complètement en le retirant de la mémoire du téléphone, puis que vous l'ouvrez à nouveau. Vous allez être sur la page de l'utilisateur. Il va avoir un bouton continué qui va vous amener sur la page créer un nouvel entrainement, s’il n’y a pas d’entrainement de créé sinon vous allez être redirigé sur la page dernier entrainement.

5) Si vous voulez supprimer l'application au complet du téléphone pour faire des tests. Il suffit de garder le doigt enfoncé sur l'app puis de sélectionner supprimer. Après il reste à la réinstaller (même chose que l'étape 1). Si vous voulez seulement supprimer l’utilisateur, vous avez juste à appuyer sur le bouton conçu pour ça.
